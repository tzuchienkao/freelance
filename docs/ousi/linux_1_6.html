Scalix 管理控制台(SAC)

Scalix 花費數年的時間發展了一套工具，允許電腦終端依他們的期望實際管理Scalix環境，沒有任何打斷核心訊息系統服務的風險。

使用 SAC，您使用瀏覽器即可管理使用者、群組、公用郵件連絡人群組、伺服器處理程序、訊息佇列在任何時間、任何地點。

  • 使用瀏覽器經由網路集中管理所有Scalix伺服器
  • 經由 Active Directory 管理使用者和群組
  • 經由設定郵件信箱大小或訊息停留時間規劃訊息保持政策
  • 針對監視及故障排除需要的事件活動記錄及分析
  • 具彈性的命令列管理介面
  • 可為自動化管理活動編列腳本
  • 當使用者超越限制時，可規劃設定約束功能
  • 與其他第三方即時監控工具相容
