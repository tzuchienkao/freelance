企業版概觀

Scalix 是一套實證過的企業用級電子郵件及行事曆平台，它基於Linux及開放系統架構 (OSA)。

Scalix 大幅降低企業訊息系統的成本及複雜度，同時增強了可靠性及適型能力。

Scalix 執行於企業工作群組模式下並完全及通透地支援各種格式電子郵件、共用目錄、 群組行事曆、公用資料夾、連絡人及工作管理等等，這些特性及功能與Microsoft Outlook 2003、XP 及 2000的使用者功能等同。

Scalix 也支援先進操作模式，譬如離線資料夾同步化，郵件規則及委派機能。

Outlook 2003 及 Scalix Enterprise Server

此外也有選項可以以Outlook做為使用者介面，使用 Scalix Web Access，公司可以現在實現Thin-Client的效益同時不會損及功能或效率。

使用 Scalix Web Access，使用者可以在任何時間及從任何地方存取他們的電子郵件及行事曆系統。
